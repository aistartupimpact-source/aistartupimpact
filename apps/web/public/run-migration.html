🚀 Run Category Migration

Add the category column to the Startup table

What this does:
  • Adds a category column to the Startup table
  • Creates an index for better performance
  • Enables category selection and display
  • Safe to run multiple times

After Migration:

  1. Go to Founder Portal
  2. Edit your startup
  3. Select "Media & Entertainment" (or any category)
  4. Click "Save Changes"
  5. Visit your startup page
  6. ✅ Your selected category will now display!
