# SEO Quick Reference Card

## 🎯 Quick Checklist for 100% SEO Score

### Title (20 points)
- [ ] 30-60 characters long
- [ ] Focus keyword in first 20 characters
- [ ] Contains power word (Best, Ultimate, Complete, Guide, etc.)
- [ ] Compelling and click-worthy

**Example:** "Best AI Startup Tools in India 2026 - Complete Guide"

### Meta Description (15 points)
- [ ] 120-160 characters long
- [ ] Contains focus keyword
- [ ] Has call-to-action (Learn, Discover, Find out)
- [ ] Compelling summary

**Example:** "Discover the best AI startup tools transforming India's tech ecosystem. Learn how founders leverage cutting-edge solutions to scale faster."

### Content (25 points)
- [ ] 1000+ words (optimal)
- [ ] Focus keyword used 5-10 times (0.5-2.5% density)
- [ ] 2-3 H2 headings with keyword
- [ ] 2-3 H3 subheadings
- [ ] Paragraphs under 100 words
- [ ] 2-3 internal/external links

### Keywords (15 points)
- [ ] Long-tail keyword (2-4 words)
- [ ] Keyword in first paragraph
- [ ] Keyword in at least one heading
- [ ] Related keywords included

### Images (10 points)
- [ ] 2-3 relevant images
- [ ] All images have alt text
- [ ] Alt text is 5-15 words
- [ ] Alt text includes keyword naturally

**Example Alt Text:** "Indian AI startup founder using machine learning tools on laptop"

### Readability (10 points)
- [ ] Sentences under 20 words average
- [ ] Transition words (however, therefore, moreover)
- [ ] Bullet points or numbered lists
- [ ] Bold text for emphasis

### Technical (5 points)
- [ ] URL slug: 3-6 words
- [ ] Slug under 60 characters
- [ ] Uses hyphens (not underscores)

**Example Slug:** "best-ai-startup-tools-india"

## 🔴 Critical Issues (Fix Immediately)

- Missing focus keyword in title
- No meta description
- Missing image alt text
- Content under 300 words
- No headings in content

---

## 🟡 Common Warnings (Should Fix)

- Title too short/long
- Keyword density too low/high
- Keyword not in first paragraph
- Missing headings
- Paragraphs too long

---

## 💡 Quick Wins (Easy Improvements)

1. **Add power word to title** (+3 points)
2. **Include CTA in meta description** (+3 points)
3. **Add bullet points** (+2 points)
4. **Use transition words** (+3 points)
5. **Add one more heading** (+2 points)

## 📸 Image Alt Text Best Practices

### ✅ DO:
- "Indian startup founder presenting AI product to investors"
- "Machine learning dashboard showing real-time analytics"
- "Team of developers coding AI startup application"

### ❌ DON'T:
- "Image" or "Photo"
- "img_1234.jpg"
- "Click here"
- Very long descriptions (>15 words)

## ⚡ 5-Minute SEO Boost

1. Set focus keyword
2. Add keyword to title (first 20 chars)
3. Write 150-char meta description with keyword

5. Create 2 H2 headings with keyword
